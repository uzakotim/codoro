## Usage

### Install Dependencies
```
cd codoro
```
### Using yarn
```
yarn 
```
### Using npm
```
npm install
```

### Running in dev mode
```
yarn run dev
```
or
```
npm run dev
```
### Build
```
yarn build
```
or
```
npm run build
```
